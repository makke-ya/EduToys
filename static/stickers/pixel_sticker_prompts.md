# シール画像生成プロンプト集

## 共通ルール
- 透過背景
- PNG 出力
- **ドット絵**
- 幼児向け知育教材に適した、やさしく親しみやすい見た目
- 物体のシルエットが一般的で、ひと目で何かわかること
- 背景・文字・フレーム・透かしなし
- 中央配置
- 小さく表示しても判別しやすいこと
- ベクター風ではなく、**本物の pixel art**
- パステル寄りの色使い
- シールらしい完成品に見えること

```text
Create a premium-quality pixel art sticker of a rabbit for children ages 1-6. Transparent background. PNG output. Use true pixel art, not vector art, not smooth illustration. Show a classic, widely recognizable rabbit silhouette with long upright ears, round head, small body, short arms and legs, and a tiny fluffy tail. Soft white fur, pink inner ears, friendly face, pastel palette, simple polished shading, readable at small size. Make it look like a collectible sticker for an educational children's app. No text, no background, no frame, no watermark.
```
