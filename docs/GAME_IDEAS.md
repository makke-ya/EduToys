# GAME_IDEAS.md - 知育ゲーム内容案と実装管理

このドキュメントは、EduToysで提供する知育ゲームのアイディアを管理し、実装状況を追跡するためのものです。
幼児教育の5領域（言葉・環境・表現・人間関係・健康）に基づき、1歳〜6歳の成長段階に合わせた100個の案を整理しています。

## 1. 知育ゲーム・カリキュラム（100案）

各ゲーム案の横にある `[ ]` は実装状況を示します（例: `[games/001_xxx]`）。

### ① 言葉（Language）
*文字への関心、語彙力、コミュニケーションの基礎を養います。*
- **L-01. 直線なぞり**: 運筆の基礎。点と点をまっすぐ繋ぐ。 [ ]
- **L-02. 曲線なぞり**: ぐるぐる、ジグザグなどの運筆練習。 [ ]
- **L-03. ひらがななぞり（あ〜ん）**: 書き順ガイド付きのなぞり書き。 [ ]
- **L-04. カタカナなぞり（ア〜ン）**: カタカナのなぞり書き。 [ ]
- **L-05. アルファベットなぞり**: A-Zのなぞり書き。 [ ]
- **L-06. 絵カード名前当て**: イラストを見て名前を4択から選ぶ。 [games/004_name_guess]
- **L-07. しりとり列車**: 正しいしりとりになるように絵を繋げる。 [ ]
- **L-08. 反対ことばパズル**: 「大きい」の反対は？などの対義語ペア。 [games/007_opposites]
- **L-09. 動詞アクション**: 「歩く」「笑う」などの動詞に合う絵を選ぶ。 [ ]
- **L-10. 形容詞マスター**: 「赤い」「長い」などの状態を表す言葉。 [ ]
- **L-11. 助詞のパズル**: 「は・を・に」を正しく入れる。 [ ]
- **L-12. 3語文メーカー**: 「（だれが）＋（どこで）＋（なにをした）」を作る。 [ ]
- **L-13. お話の順番**: 4枚の絵を物語の順に並べ替える。 [ ]
- **L-14. 読み聞かせクイズ**: 短いお話を聞いて、内容に関する質問に答える。 [ ]
- **L-15. あいさつどっち？**: 場面に合った挨拶（おはよう、ありがとう等）を選ぶ。 [ ]
- **L-16. 漢字の成り立ちパズル**: 山や川などの象形文字とイラストのペア。 [ ]
- **L-17. タイピングごっこ**: 画面のキーボードを叩いて文字を出す。 [ ]
- **L-18. 言葉の迷路**: 指定された文字（「あ」など）を通ってゴールする。 [ ]
- **L-19. 穴埋めもじ**: 抜けている一文字を当てる。 [ ]
- **L-20. 季節の言葉探し**: 春・夏・秋・冬に関連する言葉を集める。 [ ]
- **L-21. だくおん（がぎぐげご）**: 濁音の付く言葉の練習。 [ ]
- **L-22. はんだくおん（ぱぴぷぺぽ）**: 半濁音の練習。 [ ]
- **L-23. ようおん（きゃ・きゅ・きょ）**: 拗音の理解。 [ ]
- **L-24. そくおん（っ）**: 促音を含む言葉。 [ ]
- **L-25. ちょうおん（ー）**: 長音の聞き分け。 [ ]
- **L-26. ていねいご**: 「です・ます」や「お・ご」の付く言葉。 [ ]
- **L-27. でんわごっこ**: 相手の質問にボタンで答える会話練習。 [ ]
- **L-28. おてがみ作成**: スタンプや文字を選んで手紙を完成させる。 [ ]
- **L-29. じしょ引きごっこ**: 言葉を調べると絵が出る簡易辞書。 [ ]
- **L-30. はやぐちことば**: 音声に合わせてリズムよくタップする。 [ ]

### ② 環境（Environment）
*数、図形、自然、社会への関心を広げます。*
- **E-01. 数え上げ（1-10）**: リンゴをタップして数える。 [games/001_counting_apples]
- **E-02. 数字なぞり**: 1-10の数字をなぞって書く。 [ ]
- **E-03. 数つなぎ**: 数字の順に点を繋いで絵を出す。 [ ]
- **E-04. 大小比較**: どちらが「大きい/小さい」か当てる。 [games/002_compare_size]
- **E-05. 足し算の基礎**: 2個と3個を合わせるといくつ？ [ ]
- **E-06. 引き算の基礎**: 5個から2個取るといくつ？ [ ]
- **E-07. 10の合成**: あといくつで10になるか（補数）。 [ ]
- **E-08. おかいもの（硬貨）**: 1円〜100円玉を使った支払いごっこ。 [ ]
- **E-09. 時計の読み（何時）**: 短針の指す時間を当てる。 [ ]
- **E-10. カレンダーパズル**: 曜日の順序や日付の配置。 [ ]
- **E-11. 形合わせ（あかまる風）**: 色と形を指定された場所に置く。 [games/003_shape_match]
- **E-12. 図形の回転**: 回転した形が元のどれと同じか当てる。 [ ]
- **E-13. 積み木数え**: 隠れている積み木を含めて合計を出す。 [ ]
- **E-14. 複雑な迷路**: 分かれ道のある迷路。 [ ]
- **E-15. ピースパズル**: 4〜12分割程度のジグソーパズル。 [ ]
- **E-16. 線対称（はんぶんこ）**: 半分の絵から全体を推測する。 [ ]
- **E-17. 展開図（箱作り）**: 展開図からできる形を選ぶ。 [ ]
- **E-18. 重なり図形**: 2つの形が重なるとどう見えるか。 [ ]
- **E-19. かげあてクイズ**: シルエットから元の物を当てる。 [ ]
- **E-20. 地図の読み**: 目的地まで矢印で案内する。 [ ]
- **E-21. 動物の親子**: 親の動物と子の動物を繋げる。 [ ]
- **E-22. 植物の成長**: 種→芽→花→実の順に並べる。 [ ]
- **E-23. 季節の行事**: 七夕やクリスマスなどのアイテム分け。 [ ]
- **E-24. 天気と服装**: 晴れ・雨・雪に適した服を選ぶ。 [games/008_clothes]
- **E-25. こんちゅう採集**: 草むらから指定された虫を探す。 [ ]
- **E-26. 星座つなぎ**: 星を繋いで動物などの形を作る。 [ ]
- **E-27. 浮くかな？沈むかな？**: 水の中に入れた時の浮き沈み予測。 [ ]
- **E-28. じしゃく遊び**: くっつく物とくっつかない物。 [ ]
- **E-29. 色のまぜまぜ**: 赤＋青＝？などの混色実験。 [games/009_color_mix]
- **E-30. ゴミ分別**: 燃える・燃えない・リサイクルの仕分け。 [ ]

- **A-04. 粘土こねこね**: 指でなぞると形が変わる粘土遊び。 [ ]

- **A-10. 指揮者ごっこ**: 速さを変えると音楽のテンポが変わる。 [ ]
- **A-11. ダンス・フォロー**: 画面のキャラの動きを真似る。 [ ]
- **A-12. ジェスチャー当て**: アニメーションを見て何をしているか当てる。 [ ]
- **A-13. 影絵遊び**: 光と手で動物の形を作るシミュレーション。 [ ]
- **A-14. 着せ替えコーデ**: パーティーや海などのテーマに合わせる。 [ ]
