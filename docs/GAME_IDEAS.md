## 2. 共通品質ガイドライン

- **音声**: すべてのゲームに開始時の「ずんだもん」ガイド、クリア時の「よくできました」ボイスを必須とする。
- **UI**: パステルカラー（`static/common/css/game_style.css`）と、ぷっくりした3Dボタンを基本とする。
- **フィードバック**: 不正解時は「ズッコケ」音とマイルドな揺れ。正解時は「はなまる」と「キラキラ」。
